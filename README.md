# RenewPilot — Frontend

React + TypeScript + Vite frontend for RenewPilot, a multi-tenant SaaS for tracking digital infrastructure renewals (domains, SSL, servers, hosting, licenses, CDN).

## Stack

- React 18 + TypeScript
- Vite
- TanStack React Query v5
- React Router v6
- Tailwind CSS
- Recharts

## Getting Started

```bash
npm install
npm run dev
```

App runs at `http://localhost:8080`.

## Environment

Create a `.env.local` file:

```
VITE_API_URL=http://localhost:3000/api/v1
```
